# CDFViewer.jl

[![Documentation](https://img.shields.io/badge/docs-stable-blue.svg)](https://gordi42.github.io/CDFViewer.jl/)

CDFViewer is an interactive viewer for NetCDF files and zarr stores, built on
[GLMakie](https://docs.makie.org/stable/). It opens a dataset straight from
your terminal and lets you explore every variable with a command prompt, a
GUI menu, sliders, and animations — no scripts or notebooks required.

**Documentation:** [gordi42.github.io/CDFViewer.jl](https://gordi42.github.io/CDFViewer.jl/) —
a full manual with a guided tour, examples of every plot type, and a complete
command reference. All screenshots and REPL transcripts in the manual are
generated by running the application at documentation build time, so they
always match the code.

## Features

- **Quick look, zero code:** open any NetCDF file or zarr store and get a
  plot with a handful of keystrokes.
- **Command REPL:** a `CDFViewer>` prompt with tab completion, persistent
  history, and reverse search.
- **Interactive menu:** every setting is also available in a GUI window —
  both interfaces stay in sync.
- **Nine plot types:** from line plots over heatmaps and filled contours to
  3D surfaces and volume rendering.
- **Geographic projections:** maps with coastlines, land masses, and
  satellite imagery.
- **Animation and recording:** play any dimension like a movie and record it
  to MP4, MKV, WebM, or GIF.
- **Unstructured grids:** ICON-style output is interpolated on the fly;
  external grid files are found automatically.
- **Reproducible sessions:** export the current view as a command line that
  recreates it exactly.

# Installation

You need Julia installed on your system (at least version `1.12`). You can
download it from [here](https://julialang.org/downloads/).

## Running using julia

1. Clone the repository and instantiate the environment:
   ```bash
   git clone https://github.com/Gordi42/CDFViewer.jl.git
   cd CDFViewer.jl
   julia --project=. -e 'using Pkg; Pkg.instantiate()'
   ```
   The one-time instantiation precompiles the package together with a built-in
   workload that covers the whole interactive surface, so the app starts in
   roughly 10 seconds and stays smooth afterwards (no compilation pauses on
   first clicks).
2. Run the application:
   ```bash
   julia --project=/path/to/CDFViewer.jl -e 'using CDFViewer; julia_main()' <path_to_your_file.nc>
   ```

## Compiling using PackageCompiler (optional fast path)

For the fastest startup (~1-3 seconds), you can bake everything into a system
image. The trade-offs are a large system image (several hundred MB) and a long
one-time build.

1. Run the build script from the repository:
    ```bash
    ./build.sh
    ```
2. After the build is complete, an executable `cdfviewer` will be created in the `build` folder. You can run it using:
    ```bash
    ./build/cdfviewer <path_to_your_file.nc>
    ```

Optionally, you can add the `cdfviewer` executable to your PATH for easier access.

Note: the system image pins the compiled code; rebuild after updating the
package or its dependencies.

## Remote

If you want to run the application on a remote server and forward the display
to your local machine, you can use SSH with X11Trusted forwarding (`ssh -Y`
option).

# Usage

Select a variable, the plot axes, and a plot type at the prompt — the figure
window opens and updates live:

```
CDFViewer> v temperature
CDFViewer> x lon
CDFViewer> y lat
CDFViewer> p heatmap
```

Type `help` for a list of commands, or press TAB anywhere for completion.
The [manual](https://gordi42.github.io/CDFViewer.jl/) covers everything else:

- [Getting Started](https://gordi42.github.io/CDFViewer.jl/usage/getting_started/) — a guided first session.
- [Plot Types](https://gordi42.github.io/CDFViewer.jl/usage/plot_types/) — all nine plot types with examples.
- [Customizing Plots](https://gordi42.github.io/CDFViewer.jl/usage/customization/) — colormaps, labels, and map projections.
- [Animation and Playback](https://gordi42.github.io/CDFViewer.jl/usage/animation/) — play dimensions and record movies.
- [Unstructured Grids](https://gordi42.github.io/CDFViewer.jl/usage/unstructured/) — ICON output and external grid files.
- [Configuration](https://gordi42.github.io/CDFViewer.jl/usage/configuration/) — config file and environment variables.
- [Command Line Options](https://gordi42.github.io/CDFViewer.jl/reference/cli/) and
  [REPL Commands](https://gordi42.github.io/CDFViewer.jl/reference/commands/) — full reference.
